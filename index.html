JSON-to-Go-Interface

Convert JSON to Go interface

This tool instantly converts JSON into a Go interface{} notation. Paste a JSON structure on the left and the equivalent Go type will be generated to the right, which you can paste into your program. The script has to make some assumptions, so double-check the output!

For an example, try converting JSON from the SmartyStreets API or the GitHub API.
